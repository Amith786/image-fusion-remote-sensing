# Image Fusion for Improved Situational Awareness in Remote Sensing Operations

## Overview

This project focuses on enhancing situational awareness in remote sensing applications through image fusion techniques. The system combines Electro-Optical (EO) and Synthetic Aperture Radar (SAR) imagery to generate fused images with improved visual quality and information content.

## Objectives

* Fuse EO and SAR imagery for enhanced visualization.
* Improve situational awareness in remote sensing environments.
* Enhance image quality and feature representation.
* Evaluate fusion performance using image quality metrics.

## Technologies Used

* Python
* OpenCV
* NumPy
* PyWavelets
* Matplotlib
* Machine Learning
* Image Processing

## Key Features

* EO and SAR image fusion
* Multi-sensor data integration
* Image enhancement and visualization
* Terrain and scene analysis
* Performance evaluation using PSNR and SSIM

## Methodology

1. Acquire EO and SAR images.
2. Preprocess and align image inputs.
3. Apply image fusion techniques.
4. Generate fused output images.
5. Evaluate performance using quality metrics.

## Results

The fused images provide enhanced information compared to individual sensor outputs and support improved decision-making in remote sensing operations.
